# Slinnva Agenda

Deze map is alleen voor de online agenda van Slinnva Professionals.

## Links

Klantagenda:

```txt
/agenda.html
```

Beheeragenda:

```txt
/agenda-beheer.html
```

De basislink van Render opent automatisch de klantagenda.

## Render instellingen

Build Command:

```txt
npm install
```

Start Command:

```txt
npm start
```

Environment Variables:

```txt
RESEND_API_KEY=uw Resend API key
CONTACT_EMAIL_FROM=Slinnva Professionals <agenda@slinnva.com>
```

Aanvragen worden automatisch doorgestuurd naar:

```txt
info@slinnva.com
```
