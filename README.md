# Slinnva Professionals Agenda

Online agenda en interactieve workshopgids voor Samen Actief & Verbonden.

## Start

```txt
npm install
npm start
```

## Belangrijke links

Klantagenda:

```txt
/agenda.html
```

Beheeragenda:

```txt
/agenda-beheer.html
```

## Mailinstellingen

Zet deze environment variables op de hosting:

```txt
RESEND_API_KEY
CONTACT_EMAIL_FROM
```

Voorbeeld:

```txt
CONTACT_EMAIL_FROM=Slinnva Professionals <agenda@slinnva.com>
```
